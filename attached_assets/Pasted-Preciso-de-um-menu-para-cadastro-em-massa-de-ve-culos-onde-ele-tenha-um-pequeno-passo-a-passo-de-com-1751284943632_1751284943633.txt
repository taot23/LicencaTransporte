🔄 2. Processamento no Backend
Endpoint sugerido:
http
Copiar
Editar
POST /api/veiculos/importar-csv
Content-Type: multipart/form-data
Processamento:
Parse do CSV com Papaparse (frontend) ou csv-parser (backend)

Validação de campos:

Placa com formato válido

Tipo de veículo existente

CPF/CNPJ do transportador válido e existente

Veículos válidos são salvos no banco

Retorno com:

Lista de veículos inseridos

Lista de erros (linha + motivo)

Exemplo de backend (Node.js + Express + csv-parser):
ts
Copiar
Editar
router.post('/importar-csv', upload.single('arquivo'), async (req, res) => {
  const resultados = [];
  const erros = [];

  fs.createReadStream(req.file.path)
    .pipe(csvParser())
    .on('data', (linha) => {
      try {
        validarCampos(linha);
        resultados.push(formatarParaBanco(linha));
      } catch (err) {
        erros.push({ linha, erro: err.message });
      }
    })
    .on('end', async () => {
      await Veiculo.insertMany(resultados);
      res.json({ inseridos: resultados.length, erros });
    });
});
